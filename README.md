##Comandos git
- git clone(Clonar repositorio)
- git branch(mostrar a branch atual)
- git checkout -b (criar uma nova branch)
- git commit -m (salvar alteração)
- git status(verificar status da branch)
- git push origin(enviar alterações ao repositorio remoto)
- git add . (adicionar todos os arquivos do path no stage changes)
- git add arquivo(adicionar um unico arquivo no stage changes)
